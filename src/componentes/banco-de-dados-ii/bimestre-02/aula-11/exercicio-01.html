Enunciado

Neste exercício, você irá aplicar os conhecimentos adquiridos sobre os predicados LIKE, BETWEEN e IN para realizar consultas complexas em um banco de dados de streaming de filmes. Você será desafiado a filtrar dados textuais, faixas de valores e conjuntos específicos de informações.

Preparação

  1. Configuração do Ambiente: Abra seu terminal ou o MySQL Workbench e conecte-se ao seu servidor local.
  2. Execução do Script: Copie o script da seção "1. Script de Preparação" e execute-o integralmente. Verifique se o banco streaming_db foi criado com sucesso.
  3. Resolução dos Blocos:
    • Para cada item dos desafios, escreva o comando SELECT correspondente.
    • Execute comando por comando e valide se o resultado retornado condiz com o esperado pelo enunciado.
  4. Validação: Compare seus resultados com a massa de dados inserida. Por exemplo, no Bloco 1, item 1, você deve encontrar filmes como "De Volta para o Futuro" e "O Exterminador do Futuro 2".
  5. Dica: Lembre-se que o % no LIKE substitui qualquer quantidade de caracteres, enquanto o _ substitui exatamente um caractere.

1. Script de Preparação (MySQL)

Execute o script abaixo no seu ambiente MySQL para criar o banco de dados, as tabelas e popular a massa de dados necessária para os desafios.

CREATE DATABASE streaming_db;
USE streaming_db;

CREATE TABLE filmes (
    id_filme INT AUTO_INCREMENT PRIMARY KEY,
    titulo VARCHAR(150),
    ano_lancamento INT,
    categoria VARCHAR(50),
    classificacao VARCHAR(5),
    preco_aluguel DECIMAL(5,2)
);

CREATE TABLE acessos (
    id_acesso INT AUTO_INCREMENT PRIMARY KEY,
    id_filme INT,
    data_acesso DATETIME,
    usuario VARCHAR(50),
    FOREIGN KEY (id_filme) REFERENCES filmes(id_filme)
);

2. Desafios de Prática

Bloco 1: LIKE

  1. Liste todos os filmes que contenham a palavra 'Futuro' em qualquer parte do título.
  2. Liste os filmes que começam com a letra 'O' e terminam com a letra 'o'.
  3. Busque os filmes que possuem o padrão 'Toy Story' seguido de exatamente um caractere qualquer.
  4. Identifique filmes cujos títulos possuam a sequência 'de' (com um espaço antes e um depois) em qualquer posição.
  5. Selecione os filmes que possuem exatamente 4 caracteres antes do ano '2049' no título.

Bloco 2: BETWEEN

  1. Liste os filmes lançados entre os anos de 1990 e 1999.
  2. Selecione os filmes cujo preço de aluguel esteja na faixa de 2,00 a 5,00.
  3. Busque todos os acessos realizados durante o mês de Março de 2024.
  4. Identifique os filmes que não possuem preço de aluguel entre 0,00 e 3,00.
  5. Selecione os filmes cujo id_filme esteja entre 10 e 20, apresentando o resultado em ordem alfabética.

Bloco 3: IN

  1. Selecione os filmes que pertençam às categorias 'Ação', 'Policial' ou 'Suspense'.
  2. Liste os filmes que possuem classificação indicativa 'L', '10' ou '12'.
  3. Utilizando o operador IN, liste os títulos dos filmes que aparecem na tabela de acessos.
  4. Utilizando o operador NOT IN, identifique quais filmes ainda não possuem nenhum registro de acesso.
  5. Liste os acessos realizados pelos usuários 'Ana Oliveira' ou 'Ricardo Santos'.
